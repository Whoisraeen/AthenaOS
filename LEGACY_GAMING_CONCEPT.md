# LEGACY — Abandoned gaming thesis

This file replaces the old gaming-desktop manifesto that shipped with the
bootstrap codebase.

**AthenaOS does not pursue a gaming-first product.** The design bible is
[`Athena_Concept.md`](Athena_Concept.md): embodied AGI for a humanoid /
synthetic person under AthGuard.

## What was parked

| Former goal | Athena stance |
|---|---|
| “Gaming isn’t a mode” / AthPlay / GameOS couch UI | Parked — not a product surface |
| Steam / Proton / AthBridge day-one | Parked |
| Anti-cheat vendor partnerships (EAC/BattlEye) | Parked — attestation primitives may be reused for **body/safety attestation**, not game AC |
| Consumer app store economics | Parked |
| SCHED_BODY as “game threads” | Retargeted — body/control real-time class |

Historical design notes may still mention Steam, GameOS, or anti-cheat in
`docs/` and kernel comments. Treat those as **bootstrap residue**, not roadmap.
