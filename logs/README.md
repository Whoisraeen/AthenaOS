# Captured boot logs

Real AthenaOS boot transcripts pulled off hardware, kept for reference. These are
snapshots ‚Äî the live serial log is written to `%TEMP%\raeen-serial.log` by xtask
(never the repo, per CLAUDE.md ¬ß7), and the on-device ring is persisted to
`BOOTLOG.TXT` on the boot media (see `kernel/src/bootlog_persist.rs`).

All captures below are from **Athena** (Beelink EliteMini, Ryzen 5 7640HS) via
the internal-NVMe ESP fallback: booted with no USB stick, safe-mode enabled, so
the kernel wrote the RAM ring into the pre-allocated 1 MiB `BOOTLOG.TXT` on the
internal EFI System Partition. Extracted read-only via
`scripts/read-bootlog.ps1 -Disk 0`.
